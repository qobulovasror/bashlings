# 💡 pipe1 — Hints

## Step 1
The `>` operator — **overwrites** a command's output into a file. If the file does not exist, it is created.

```bash
echo "matn" > fayl.txt
```

## Step 2
To print a file's contents use `cat <fayl>`.

## Step 3
Write the two commands on separate lines or chain them with `&&`.
