# 💡 pipe1 — Maslahatlar

## 1-bosqich
`>` operatori — buyruq natijasini faylga **qayta yozadi**. Fayl yo'q bo'lsa, yaratiladi.

```bash
echo "matn" > fayl.txt
```

## 2-bosqich
Fayl mazmunini chiqarish uchun `cat <fayl>` ishlatiladi.

## 3-bosqich
