# 💡 pipe2 — Maslahatlar

## 1-bosqich
`>` va `>>` farqi:
- `>`  — fayl mavjud bo'lsa, **eski mazmun o'chiriladi** va yangidan yoziladi
- `>>` — fayl mavjud bo'lsa, **oxiriga qo'shib boriladi** (append)

## 2-bosqich
Sizga `>>` kerak — chunki "birinchi" qator saqlanishi kerak.

## 3-bosqich
Qo'shgandan keyin `cat` orqali fayl mazmunini chiqaring.

## ✅ Yechim
```bash
echo "ikkinchi" >> log.txt
cat log.txt
```
