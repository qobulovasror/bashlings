# 💡 pipe4 — Maslahatlar

## 1-bosqich
`seq` — sonlar ketma-ketligini chiqaradi.

```bash
seq 1 5
# 1
# 2
# 3
# 4
# 5
```

## 2-bosqich
`tail -n N` — input'ning **oxirgi N qator**ini chiqaradi.

```bash
seq 1 10 | tail -n 3
```

## 3-bosqich
Pipe operatori `|` bilan ulang.

## ✅ Yechim
```bash
seq 1 10 | tail -n 3
```
