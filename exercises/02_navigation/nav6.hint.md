# 💡 nav6 — Maslahatlar

## 1-bosqich
`cp` ikkita argument oladi: **manba** va **maqsad**.
```bash
cp manba.txt nusxa.txt
```

## 2-bosqich
Fayl mazmunini chiqarish uchun `cat <fayl>`.

## 3-bosqich
Ikki buyruqni alohida qatorda yozsangiz bo'ladi yoki `&&` bilan zanjirlash mumkin.

## ✅ Yechim
```bash
cp manba.txt nusxa.txt
cat nusxa.txt
```
