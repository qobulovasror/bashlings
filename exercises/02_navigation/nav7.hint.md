# 💡 nav7 — Maslahatlar

## 1-bosqich
`mv` bir buyruqda ikki ish bajaradi:
- **Ko'chirish**: `mv fayl.txt /tmp/` — boshqa katalogga
- **Rename**: `mv eski.txt yangi.txt` — nomini o'zgartirish

## 2-bosqich
`mv` katalog uchun `-r` flagi talab qilmaydi.

## ✅ Yechim
```bash
mv eski.txt yangi.txt
ls
```
