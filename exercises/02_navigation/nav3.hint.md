# 💡 nav3 — Maslahatlar

## 1-bosqich
`mkdir` bir vaqtda bir nechta argumentni qabul qiladi. Misol: `mkdir a b c` — 3 ta katalog yaratadi.

## 2-bosqich
`-p` flagi — agar yo'lda yo'q kataloglar bo'lsa, ham yaratadi. Va katalog allaqachon mavjud bo'lsa, xato bermaydi.

## 3-bosqich
Sizning vazifangiz: `docs`, `src`, `tests` — 3 ta katalog. `ls` ularni alphabet bo'yicha chiqaradi.

## ✅ Yechim
```bash
mkdir -p docs src tests
```

`ls` allaqachon skript ichida — yana yozish kerak emas.
