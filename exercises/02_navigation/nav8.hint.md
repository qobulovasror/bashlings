# 💡 nav8 — Maslahatlar

## 1-bosqich
`rm <fayl>` — o'chiradi. **Ehtiyot bo'ling** — `rm` tiklab bo'lmaydigan o'chiruv. `Trash` ga tushmaydi.

## 2-bosqich
Bir fayl o'chirib, qolganlarini ko'rishingiz kerak. `ls` argumentsiz joriy katalogni listing qiladi.

## 3-bosqich
**Diqqat:** `rm tmp*.txt` 3 ta faylni ham o'chirardi. Bizga faqat **tmp2.txt** kerak — wildcard ishlatmang, aniq nomi bilan o'chiring.

## ✅ Yechim
```bash
rm tmp2.txt
ls
```
