# 💡 nav2 — Hints

## Step 1
`cd` accepts the `~` (tilde) symbol as an argument — it is a shortcut for your **home directory**. `cd ~` and `cd $HOME` work the same way.

## Step 2
After moving, use `pwd` to print the current directory path.

## Step 3
Chain the two commands with `&&` — the second runs only if the first succeeds.
