# 💡 intro5 — Hints

## Step 1
In Bash you use a variable inside text directly within double quotes:
```bash
echo "Salom, $USER!"
```

## Step 2
You need 3 variables:
- `$USER` — username
- `$HOME` — home directory
- `$PWD` — current directory (Print Working Directory)

## Step 3
Write the first line like this:
```bash
echo "Foydalanuvchi: $USER"
```
The other two lines follow the same shape.
