# 💡 intro5 — Maslahatlar

## 1-bosqich
Bashda matnda o'zgaruvchini ishlatish — to'g'ridan-to'g'ri qo'shtirnoq ichida:
```bash
echo "Salom, $USER!"
```

## 2-bosqich
Sizga 3 ta o'zgaruvchi kerak:
- `$USER` — foydalanuvchi nomi
- `$HOME` — home katalog
- `$PWD` — joriy katalog (Print Working Directory)

## 3-bosqich
Birinchi qatorni shunday yozing:
```bash
echo "Foydalanuvchi: $USER"
```
Qolgan ikkita qator ham xuddi shu shaklda.
