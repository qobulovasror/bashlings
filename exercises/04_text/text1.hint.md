# 💡 text1 — Maslahatlar

## 1-bosqich
`head` buyrug'i — faylning **boshini** (dastlabki qatorlarini) chiqaradi.
Sukut bo'yicha 10 qator chiqaradi.

## 2-bosqich
Qator sonini boshqarish uchun `-n` flagi:
```bash
head -n 5 fayl.txt   # birinchi 5 qator
```

Yoki qisqartirilgan:
```bash
head -5 fayl.txt
```

## ✅ Yechim
```bash
head -n 3 log.txt
```
