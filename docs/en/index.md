## Why this tutorial?

For many Uzbek developers, the terminal and Bash still remain "mysterious" territory. This tutorial is aimed precisely at filling that gap — we have combined the three pillars of **theory + practice + error analysis**.

::: tip Who is it for?
- **Beginners** who are just starting to learn Linux
- **Junior developers** entering the DevOps path
- **Engineers** working with server administration
- Anyone who wants to automate their own workflows
:::

## Tutorial structure

The tutorial consists of **3 large parts** — **16 chapters** and **101 interactive exercises** in total.

### 📘 Part 1 — Linux & Bash fundamentals (32 exercises)
Designed for beginners. The journey from the terminal all the way to writing your first Bash script.

[**→ Start Part 1**](/en/part1/01-introduction)

### 📗 Part 2 — Advanced Bash Scripting (28 exercises)
Functions, arrays, `sed`/`awk`, signals, and production-grade scripts.

[**→ Go to Part 2**](/en/part2/01-functions)

### 📕 Part 3 — Real-world skills (41 exercises)
`curl`, SSH, `jq`, cron, Docker, GitHub Actions — the core skills on the DevOps path.

[**→ Go to Part 3**](/en/part3/01-network)
